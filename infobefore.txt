
OOHG -an acronym of Object Oriented (x)Harbour GUI- is an, easy to use, completely free, object oriented xBase development system for Windows, based on these tools:

* OOHG libraries and preprocessor directives for defining GUI components.
* OOHG IDE+ for defining windows interactively.
* (x)Harbour Compiler for compiling xBase code into C code
* C Compiler (MinGW or BCC) for compiling C code and linking into an exe.
* Project support site at https://oohg.github.io
* User support mail list at <https://groups.google.com/d/forum/oohg>
